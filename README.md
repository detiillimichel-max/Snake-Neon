# Snake Neon

Um Snake arcade com visual neon, controles por swipe e botões, e power-up de câmera lenta. 100% HTML/CSS/JS, sem dependências.

## 🎮 Jogar

Abra `snake-neon.html` no navegador ou publique via GitHub Pages.

## ✨ Recursos

- **Visual neon**: cobra ciano com glow, paredes iluminadas, comida magenta pulsante
- **Controles mobile**: swipe para mudar direção + botões ⬅️➡️ para virar
- **Power-up Slow-Mo**: pílula azul congela o tempo por 5s
- **Score e Recorde**: salvo em `localStorage`
- **HUD**: FPS, badge de slow-mo, score/best
- **Telas**: Start, Game Over com novo recorde
- **Responsivo**: canvas se adapta, 60 FPS

## 🕹️ Como jogar

1. Toque em **PLAY**
2. Deslize na direção desejada ou use os botões laterais para virar esquerda/direita
3. Coma os orbs magenta para crescer e ganhar +10 pontos
4. Pegue a pílula azul para ativar câmera lenta por 5 segundos
5. Evite bater nas paredes neon ou em si mesmo

## 🎯 Controles

- **Swipe**: muda direção absoluta
- **Botão ⬅️**: vira à esquerda
- **Botão ➡️**: vira à direita
- **Teclado**: A / ← vira esquerda, D / → vira direita, Espaço inicia
